User Interviews

โดยการสัมภาษณ์จะมีการเตรียมคำถามไปเป็นข้อ แต่จะเป็นการสัมภาษณ์แบบพูดคุยเพื่อที่จะให้ User ได้เล่าเรื่องพูดคุย ได้อย่างเป็นธรมชาติ และจะมีคำถามที่ได้เตรียมไว้ถามในจุดที่สำคัญและจุดที่ต้องหารเน้นย้า

ตัวอย่างคำถามที่ใช้ในการพูดคุยและสัมภาษณ์

1.เคยได้ยินชื่อสถานที่ท่องเที่ยวนี้อยู่บ้างไหม
2.รู้จักอะไรเกี่ยวกับสถานที่ท่องเที่ยวนี้อยู่บ้าง
3.คิดว่าสถานที่ท่องเที่ยวนี้น่าสนใจไหม
4.อยากไปเที่ยวสถานที่ท่องเที่ยวนี้อยู่บ้างไหม
5.ทำไมถึงอยากหรือไม่อยากไปเที่ยวสถานที่ท่องเที่ยวนี้
ตัวอย่างคำถามที่ใช่ในการพูดคุยและ สัมภาษณณ์

สรุปคำตอบที่เป็นข้อมูลสำคัญ

สรุปคำตอบที่เป็นข้อมูลสำคัญ .อยากทราบเวลาเปิด-ปิด ของสถานที่ท่องเที่ยวอยากให้ชี้แจงคำแนะนำ เช่น
ข้อควรระวังหรือข้อห้ามในสถานที่ท่องเที่ยวอยากให้บอกข้อมูลทั่วไป เช่น ชื่อสถานที่ ที่ตั้ง เวลาการเดินทาง ไม่เคยใช้แอพมาก่อนเพราะกลัวควยุ่งยากในการค้นหาสถานที่
